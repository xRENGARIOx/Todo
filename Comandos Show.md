
# Checar Resumen Interfaces
```ciscoIOS
show ip interface brief
```

# Checar Resumen Interfaces IPv6
```ciscoIOS
show ipv6 interface brief
```

# Checar Alguna Interfaz 
```ciscoIOS
show interface g0/0
```

# Checar Interfaces en Running
```ciscoIOS
show running | interface
```

# Checar IPv6
```ciscoIOS
show ipv6 int gi0/1
```
